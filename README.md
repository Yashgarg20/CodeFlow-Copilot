# 📖 Overview

CodeFlow Copilot is an **AI-powered Software Engineering Assistant** that helps developers automate repetitive software engineering tasks using **Google Gemini AI**.

Instead of switching between multiple tools for debugging, documentation, code review, repository analysis, and commit generation, CodeFlow Copilot provides all these functionalities through one intelligent web application.

The project is developed using **React.js**, **FastAPI**, **SQLite**, and **Google Gemini AI**, offering a modern, responsive, and scalable software engineering platform.

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/Yashgarg20/CodeFlow-Copilot.git
```

```bash
cd CodeFlow-Copilot
```

---

# 🔑 Backend Setup

Navigate to backend

```bash
cd backend
```

Create a `.env` file

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run backend

```bash
uvicorn main:app --reload
```

---

# 🎨 Frontend Setup

Navigate to frontend

```bash
cd frontend
```

Install packages

```bash
npm install
```

Start React

```bash
npm run dev
```
